User roles

We have 3 roles that a player can get, starting from Newcomer, then receiving the Member role, and finally getting the Trusted role. These roles are obtained sequentially (e.g. cannot go from Newcomer to Trusted), and each role has their own set of requirements like playtime and advancements. The Foxy role is the only one that cannot be obtained automatically, as it's a role reserved for those who support the server or has received it as a prize.

The first role that a new player will receive. From here you can get promoted to Member, then Trusted.

Commands

Required argument - <>
Optional argument - []

  • /afk - Toggles your AFK status.
  • /ar check [path] - Check the status on your role path.
  • /ar leaderboard [type] - View a list of players with the highest playtimes. Types include daily, weekly, monthly, and total.
  • /ar times [player] - View how much playtime a specific member has.
  • /ar track <id> <path> - Checks how far along you are with a requirement. Get the requirement id with /ar check.
  • /book [title [name]] - Edit the title or contents of a book you've written yourself.
  • /discord - Get the Discord invite.
  • /discord linked - Verify link status.
  • /durabilityping - Toggle the ping sound effect for when you hit a certain durability threshold.
  • /gc - Get info on the server including uptime, TPS, memory usage, and worlds loaded.
  • /help [search term] [page] - Get help about commands on various plugins.
  • /list [group] - Get the full list of players.
  • /mail [read|clear|clear [number]|send [to] [message]|sendtemp [to] [expire time] [message]] - Read, send, or clear mail. You can also send a mail that expires after a certain amount of time.
  • /motd - Repeats the message that appears when you log into the server.
  • /msg <to> <message> - Privately sends a message to someone.
  • /msgtoggle [on|off] - Toggles whenever you wish to receive messages or not. Keep in mind that staff can bypass this toggle.
  • /near - View all players nearby in a 100 block radius.
  • /ping - Pong!
  • /pcoords - Get the equivalent nether portal coordinates on either the overworld or nether based off where you're standing.
  • /realname <nickname> - Get the real name of a nicknamed member.
  • /r <message> - Replies to the last member that has sent or received a message to/from yourself.
  • /rtoggle [on|off] - Toggles whenever you want to reply to the last message recipient.
  • /rules [chapter] [page] - View the ZorroMC rules.
  • /seen <player> - View how long has a player been online or offline.
  • /skull <player> - Change the username of a skull you've obtained, either via a mob head or player head.
  • /togglehud - Toggles a HUD that shows your current coordinates, direction, and time.
  • /togglehud - Accept a staff member's request to teleport to your location.
  • /tpdeny - Deny a staff member's request to teleport to your location.
  • /tps - View the current and average server TPS.
  • /vdt status - Get the current view distance of each world.
  • /warnings - View any warnings that you may have received.

Obtained after 4 hours of playtime, 3 days since first joining the server, and the following advancements: Sweet Dreams, Monster Hunter, Acquire Hardware, Stone Age.

Commands

Required argument - <>
Optional argument - []

Obtained after 15 hours of playtime, 14 days since first joining the server, and the following advancements: Diamonds!, Return to Sender, Enchanter, Free the End.

Commands

Required argument - <>
Optional argument - []

  • /disguise [entity] - Disguise yourself with a wide variety of entities. Run the command without any arguments to get the list of available entities.
  • /dviewself - Toggles whenever you want to view your own disguise or hide it from your view. (others will still see your disguise)
  • /undisguise - Undisguises yourself.
  • /pp - Opens a GUI to set particle effects.
  • /hat [remove] - Put on any block as a hat. Best used with banners.
  • /nick <nickname|off> - Set any nickname with color codes on yourself.
